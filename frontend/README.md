# Frontend Web UI

React application built with Vite for drawing network graphs and visualizing algorithm execution.

## Setup

```bash
npm install
npm run dev
```

## Tech Stack
- **React + Vite**: UI and fast development server.
- **React Flow**: Core library for rendering interactive graph nodes and edges.
- **Zustand**: Global state management (graph data, execution logs).
- **TailwindCSS**: Styling and utility classes.

*Note: Ensure backend is running locally to fetch algorithm results.*
